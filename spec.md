# Online Shop

## Current State

Интернет-магазин лампочек с корзиной, заказами, балансом кошелька, чатом с продавцом и Премиум-подпиской. Баланс хранится в localStorage. На странице баланса (`/balance`) пользователь может самостоятельно пополнять счёт кнопками +$5/+$10/+$20/+$50. Числового ID у пользователя нет.

## Requested Changes (Diff)

### Add
- Числовой ID (только цифры, авто-генерация при первом посещении) для каждого пользователя — хранится в localStorage, отображается на странице баланса.
- Страница `/admin/balance` — администратор вводит числовой ID пользователя и сумму, пополняет его кошелёк. Баланс хранится по userId в localStorage.
- Ссылка "Кошельки пользователей" в AdminLayout.

### Modify
- `useBalance.ts` — баланс привязан к числовому userId (ключ `shop_balance_<userId>`), убрана функция `addBalance` из публичного интерфейса (пополнение только через AdminBalancePage).
- `BalancePage.tsx` — убрать кнопки "Быстрое пополнение" и кнопку "Оплатить Премиум с баланса через пополнение"; оставить только просмотр баланса, ID пользователя и историю операций. Кнопка оплаты Премиум с баланса остаётся если у пользователя достаточно средств.

### Remove
- Кнопки самостоятельного пополнения баланса на странице `/balance`.
