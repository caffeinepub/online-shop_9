# Online Shop

## Current State
- Интернет-магазин с каталогом товаров, корзиной и оформлением заказов
- Авторизация через Internet Identity (роли: admin, user, guest)
- Хранилище изображений (blob-storage)
- Пользователи могут просматривать товары и оформлять заказы
- Панель администратора с паролем (frontend-only защита через /admin)
- Нет чата между покупателями и продавцами
- Нет системы премиум-подписки
- Нет лимита на количество товаров

## Requested Changes (Diff)

### Add
- **Премиум-подписка**: кнопка с короной в шапке, Stripe оплата, $5 / 30 дней
- **Лимит товаров**: обычный пользователь (seller) может добавить не более 10 товаров; при достижении лимита предлагать купить премиум
- **Чат**: на странице каждого товара покупатель может написать продавцу (владельцу товара); история сообщений хранится в бэкенде
- **Премиум-статус**: в бэкенде хранить дату истечения премиума для каждого пользователя; после оплаты через Stripe активировать на 30 дней

### Modify
- Продукт теперь имеет поле `sellerId` (Principal) — кто добавил товар
- Функция `createProduct` проверяет лимит (10 товаров для не-премиум продавцов)
- Навигация: добавить кнопку "Премиум" с иконкой короны справа в шапке

### Remove
- Ничего не удаляется
